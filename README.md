## 1. Data
229 TNOs: [marsset_data_full.csv](https://github.com/ZehaoJin/causalTNOs/blob/main/marsset_data_full.csv)
- a : semimajor axis
- e : eccentricity
- inc : inclination
- c : spectral slope
- class2 : population class, `{'cen':'Centaurs','cla':'Classicals','det':'Detached','res':'Resonant','sca':'Scattered'}`
